A5.sln
1. open A5.sln
2. build as Debug mode
3. Open A5.exe/Debug/A5Framework

A5.exe
1. open A5.exe
2. there's two modes : create, pick and move
3. If you pick create mode, you can choose 4 shapes : circular pillar, triangle, rectangle, convex polygon
- circular pillar : pick center(1st click), pick radius(2nd click)
- triangle : pick 3 vertices(1,2,3rd clicks), you can only pick counter-clockwise ordered
- rectangle : pick left upper vertex(1st click) and pick right below vertex(2nd click)
- convex polygon : pick vertices(clicks), you can only pick counter-clockwise ordered and convex,
		if you complete the polygon, you click right mouse button to end.
